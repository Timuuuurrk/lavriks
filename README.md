# Lavrik Telegram Bot


P.S. На данный момент бот по адресу не работает ввиду отключения хостинга на него. При возобновлении работы бота об этом будет сказано в ближайшем релизе.

## Описание проекта

Telegram-бот на Python, разработанный с использованием библиотеки `pyTelegramBotAPI`.

Бот предназначен для учебных и развлекательных задач: отправляет учебные материалы, показывает расписание, предоставляет информацию о БВИ в вузы, присылает стикеры, определяет погоду в городе и отвечает на различные текстовые команды.

## Функциональность

Бот поддерживает следующие возможности:

- обработка команд Telegram
- inline-меню с кнопками
- отправка учебных материалов по физике и математике
- отправка расписания
- информация о БВИ для разных вузов
- получение погоды по названию города
- отправка случайных стикеров, фото, GIF и видео
- генерация случайной вероятности события
- получение ID пользователя, сообщения и чата
- удаление сообщений по ID

## Команды бота

| Команда | Описание |
|---|---|
| `/start` | Запуск бота и приветственное сообщение |
| `/help` | Список возможностей бота |
| `/commands` | Открытие inline-меню с разделами |
| `/weather city` | Получение погоды в указанном городе |
| `/info` | Информация о боте и ссылка на исходники |

## Установка и запуск

Склонировать репозиторий:

```bash
git clone https://github.com/Timuuuurrk/lavriks.git
```

Перейти в папку проекта:

```bash
cd lavriks
```

Установить зависимости:

```bash
pip install -r requirements.txt
```

Запустить бота:

```bash
python main.py
```

## Настройка переменных

Для работы бота нужны:

- Telegram Bot Token
- OpenWeatherMap API Key

Рекомендуется хранить токены не в коде, а в переменных окружения или `.env` файле.

Пример `.env`:

```env
BOT_TOKEN=your_telegram_bot_token
WEATHER_API_KEY=your_openweathermap_api_key
```
